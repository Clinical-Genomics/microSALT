# microSALT
WIP repo of a MLST pipeline.

## Dependencies
* Python 3.6
* BLAST
* SpaDES
* MySQL

## Database info
An additional mysql.yml file is required to be present in the microSALT sub-folder.
Formatting of the file is as follows:
```user: USER
pw: PASSWORD
host: DATABASE_URL
port: PORT
db: DATABASE_NAME```
